# MiroFish - Churn Prediction Microservice

Python FastAPI microservice for predicting customer churn (evasão) for Barbearia VIP.

## Features

- Churn prediction using XGBoost model
- Customer segmentation by profile and frequency
- Visit tracking across multiple units
- Email and WhatsApp automated actions
- REST API with health checks

## API Documentation

Once the service is running, visit:
- Swagger UI: `http://localhost:3022/docs`
- ReDoc: `http://localhost:3022/redoc`
